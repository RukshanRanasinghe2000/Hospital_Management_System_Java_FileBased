------Packages-----

Entity - Transfer encapsulated data to the requested class or method. I used getters and setters to access the encapsulated data objects.

Dao (Data Access Object)- These files are created in connection (data retrieve and save) with CSV files. 
				  *** I used java. util.Scanner for data writing and reading from CSV files.

-----Driver class-----

AppMain - This class is the driver program and interacts with the user.

data_store - All CSV files are stored within this file.
